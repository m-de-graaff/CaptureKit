# Assets Directory

This directory contains project assets such as:

- Icons and branding materials
- UI resources (images, fonts)
- Sample media files
- Documentation assets

## Structure

```
assets/
├── icons/          # Application icons
├── branding/       # Logo and brand assets
├── ui/            # UI resources
└── samples/       # Sample media files
```

## Adding Assets

When adding new assets:

1. Use appropriate file formats (SVG for icons, PNG/JPG for images)
2. Optimize file sizes for distribution
3. Include source files when possible
4. Document licensing and attribution
